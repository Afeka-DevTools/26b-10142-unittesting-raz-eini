## תקציר השיחה

### טסטים שנוספו

- `add`:
  - טסטים למספרים חיוביים, שליליים, אפס וערכים גדולים.
- `isPrime`:
  - טסטים למספרים שליליים, 0, 1, מספרים ראשוניים ומרוכבים.
- `reverse`:
  - טסטים למחרוזות ריקות, תו בודד, רווחים, מספרים, תווים מיוחדים, פלינדרומים וטאבים.
- `factorial`:
  - טסטים לחריגות בקלט שלילי וערכי פקטוריאל מ-0! עד 12!.
- `isPalindrome`:
  - טסטים לרווחים, פיסוק, אותיות גדולות/קטנות, מספרים, מחרוזות ריקות ותווים מיוחדים בלבד.
- `fibonacciUpTo`:
  - טסטים לחריגות, 0, 1, ערכים שמספר פיבונאצ'י, ערכים בין פיבונאצ'ים ומגבלות גדולות.
- `charFrequency`:
  - טסטים למחרוזת ריקה, תווים חוזרים, רווחים, מספרים, תווים מיוחדים, רגישות לאותיות, טאבים ושורות חדשות.
- `isAnagram`:
  - טסטים לאנגרמות פשוטות, מחרוזות ריקות, רווחים, רגישות לאותיות, מספרים, פיסוק, אורכים שונים ותווים חוזרים.
- `average`:
  - טסטים לחריגות, ערך יחיד, חיובי, שלילי, תערובות, אפסים, תוצאות שבריות, מספרים גדולים ועקביות.
- `filterEvens`:
  - טסטים לרשימות ריקות, רק זוגיים, רק אי זוגיים, תערובות, מספרים שליליים, אפסים, כפילויות ושמירת הסדר.
- `mostCommonWord`:
  - טסטים לריבוי תווים, פיסוק, רגישות לאותיות, רווחים, טאבים, שורות חדשות, שוויון תדירויות וסמלים מיוחדים.

### בדיקה

- הרצה של כל הטסטים בפקודה:
  ```powershell
  .\gradlew test --console=plain
  ```
- הפעלת דוח כיסוי קוד של JaCoCo בפקודה:
  ```powershell
  .\gradlew jacocoTestReport
  ```
